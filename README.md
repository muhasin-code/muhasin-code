A full-stack, multi-role inventory management system built for and actively deployed within

### 🦽 [Assisto — Smart Wheelchair Monitoring System](https://github.com/muhasin-code/assisto)
> **Flask · Socket.IO · ESP32 · WebSocket · Leaflet.js · SQLite**

A real-time IoT web application that bridges an ESP32-based smart wheelchair to a live
web dashboard — streaming GPS, health vitals, and fall detection alerts over WebSockets.

- Bidirectional WebSocket bridge between ESP32 hardware and browser clients
- Live GPS tracking (Leaflet.js), heart rate monitoring (Chart.js), fall detection alerts
- Trip history logging, user authentication, per-user device binding
- Built-in device simulator (`wheelchair.py`) for development without physical hardware
- systemd-deployable on Linux
